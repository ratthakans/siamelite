# Siam Elite Consulting — Website

พรีเมียม landing site (2 ภาษา EN/TH) สำหรับ Siam Elite Consulting
โทนน้ำเงินเข้ม + ทอง สไตล์ **startup / liquid glass** — พื้นหลัง gradient mesh
ต่อเนื่องทั้งหน้า (ไม่ตัดเป็นบล็อกสีทึบสลับกันแบบเดิม) การ์ด/แถบต่างๆ ใช้เอฟเฟกต์
กระจกฝ้า (frosted glass: `backdrop-filter: blur()` + พื้นโปร่งแสง) ลอยอยู่บนพื้นหลังนั้น

## เปิดดูเว็บ
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย (ดับเบิลคลิก) — เป็น static site ไม่ต้องติดตั้งอะไร

## โครงสร้างไฟล์
```
index.html            หน้าเว็บหลัก (ทุก section อยู่ในนี้)
assets/css/styles.css ธีม/ดีไซน์ทั้งหมด (สี ปรับที่ตัวแปร :root ด้านบน)
assets/js/app.js      สลับภาษา / เมนูมือถือ / ปุ่มลอย / ฟอร์มทีละขั้น
assets/img/logo.svg   โลโก้ SVG สำรอง (ทำเลียนแบบของจริง)
```

## 🎨 ธีมสี
ยึดสัดส่วน **ขาว 60% · navy 30% · ทอง 10%** — พื้นหลังขาวเป็นหลัก, navy ใช้กับ footer / แถบแลกเงิน / หัวตาราง / ตัวหนังสือหัวข้อ, ทองเป็น accent + ปุ่ม CTA สำคัญ
ปรับสีได้ที่ตัวแปร `:root` ด้านบนของ `assets/css/styles.css`
ค่าเริ่มต้นเป็น **ภาษาไทย** (ผู้เข้าชมกดสลับ EN ได้ที่มุมขวาบน)

## ⚠️ สำคัญที่สุด: รูปอสังหาฯ ตอนนี้เป็นภาพสต็อกตัวอย่าง ไม่ใช่ทรัพย์จริง
Section "อสังหาริมทรัพย์" (`#property`) ตอนนี้เป็น **ระบบ filter ที่ทำงานได้จริง** (กรองตาม
ประเภท/ชนิดทรัพย์/ทำเล/ห้องนอน/งบประมาณ/คำค้นหา — ทั้งหมดเป็น JavaScript ล้วน ไม่ต้องมี backend)
แต่ทั้ง 6 รายการที่ใส่ไว้ (ชื่อ, ราคา, ทำเล และ**รูปภาพ**) เป็นข้อมูล **ตัวอย่าง/สต็อกภาพ** ทั้งหมด
(ภาพจาก Unsplash ใช้เชิงพาณิชย์ได้ฟรีตาม Unsplash License แต่**ไม่ใช่ภาพบ้าน/คอนโดจริงของคุณ**)

**ห้ามปล่อยเว็บขึ้นจริงโดยไม่เปลี่ยนเป็นรูปทรัพย์จริง** — การใช้ภาพสต็อกแทนภาพทรัพย์จริงในธุรกิจ
อสังหาฯ อาจทำให้ลูกค้าเข้าใจผิดว่าเป็นทรัพย์ที่มีอยู่จริง ซึ่งเป็นความเสี่ยงด้านความน่าเชื่อถือและ
กฎหมายคุ้มครองผู้บริโภค ก่อนเปิดเว็บให้ลูกค้าจริงเห็น ให้แทนที่ไฟล์ในโฟลเดอร์
`assets/img/properties/` ด้วยรูปถ่ายทรัพย์จริงของคุณ (ชื่อไฟล์เดิมไว้ได้ หรือแก้ `src` ใน `index.html`
ที่ section `id="property"`) และแก้ชื่อ/ราคา/ห้องนอน/data-attributes ให้ตรงทรัพย์จริง

## ✅ สิ่งที่ต้องแก้ให้เป็นข้อมูลจริง (ไล่ทีละจุด)
1. **โลโก้จริง** — ตอนนี้ใช้โลโก้ SVG ตัวยืน 2 ไฟล์:
   - `assets/img/logo.svg` — สี navy+ทอง พื้นใส (ใช้บนพื้นขาว: header)
   - `assets/img/logo-gold.svg` — สีทอง พื้นใส (ใช้ทั้ง header และ footer ตอนนี้)
   เอาไฟล์โลโก้มังกรจริงของคุณ (แนะนำเป็น **PNG พื้นหลังใส**) มาวางแล้วแก้ path ใน `index.html`
   หรือทับไฟล์ `.svg` เหล่านี้ได้เลย — มีสคริปต์ช่วยทำสี/ลบพื้นหลังอัตโนมัติที่
   `assets/tools/make_logo.py` (รัน `python3 make_logo.py <ไฟล์โลโก้ต้นฉบับ>`)
2. **รูปอสังหาฯ** — ดูหัวข้อด้านบน (สำคัญมาก)
3. **เบอร์โทร / Line / WhatsApp / WeChat** — ค้นหา `+66000000000`, `@siamelite`,
   `wa.me/66000000000`, `hello@siamelite.co` แล้วแทนด้วยของจริง (อยู่ใน header, footer, ปุ่มลอย)
4. **เลขทะเบียนบริษัท** — แก้ `Company Reg. No. 0000000000000` ใน section About
5. **Google Map** — ในกล่อง "Our Office" (`.about-media`) ตอนนี้ใช้ภาพวัดเชียงใหม่ (Unsplash)
   เป็นพื้นหลังประกอบบรรยากาศชั่วคราว แทนที่ด้วย `<iframe>` แผนที่จริงของสำนักงานคุณ
6. **รีวิวลูกค้า** — เปลี่ยนข้อความ testimonial เป็นรีวิวจริง (ยิ่งมีรูปคู่ทีมงานยิ่งดี)

## 🔎 ระบบ filter อสังหาฯ ทำงานอย่างไร
ทุกการ์ดใน `.prop-grid` มี `data-status`, `data-type`, `data-location`, `data-beds`, `data-tier`
กำกับอยู่ — ตัวกรองใน `assets/js/app.js` (ฟังก์ชัน `applyFilters`) จะซ่อน/แสดงการ์ดตามค่าที่เลือก
แบบ real-time ไม่ต้องกด "ค้นหา" เพิ่มทรัพย์ใหม่ทำได้โดย copy การ์ดใน `index.html` แล้วแก้
data-attributes + เนื้อหาให้ตรง ระบบจะกรองถูกต้องทันทีโดยไม่ต้องแก้ JS เพิ่ม

> หมายเหตุ: บริการ **แลกเงิน (Currency Exchange) ถูกถอดออกชั่วคราว** (รอใบอนุญาต) — เว็บตอนนี้เน้น
> วีซ่า + อสังหาฯ/แม่บ้าน โค้ด CSS ของ section แลกเงินยังเก็บไว้ในไฟล์ พร้อมเปิดกลับเมื่อพร้อม
